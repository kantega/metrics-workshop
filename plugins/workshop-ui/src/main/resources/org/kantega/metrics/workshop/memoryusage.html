Charting memory usage now

Your task is to add a graph to the Stats page.

The Stats page shows an example graph using Highcharts. To simplify creation of graphs the page uses AngularJS with an Hightchart AngularJS directive. You task is to create a new graph on this page. Start by looking at the source in the metrics-ui plugin. Don't worry if you have never tried AngularJS, as we will mostly be writing pure Javascript or using the Highcharts API

The graph should show Java heap.used from metrics using an Angular Gauge .
